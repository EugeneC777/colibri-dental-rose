# Colibri Dental — сайт клиники

Лаконичный люксовый одностраничный сайт авторской стоматологической клиники
**Colibri Dental** (Москва): эстетика, виниры нового поколения, передовое
оборудование.

## Структура

- `index.html` — весь сайт (HTML + CSS + JS в одном файле)
- `assets/` — изображения (логотип, фото, оптимизированные для веба)

## Запуск локально

Просто откройте `index.html` в браузере, либо поднимите статический сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```
